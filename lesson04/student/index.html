for文で繰り返し処理を書こう
0〜25分
📖
ループ処理は同じ処理を繰り返すための仕組みです。for文が最も基本的な書き方です。
script.js（for文の基本）
// for（初期化; 条件; 更新）{ 処理 } for (let i = 0; i < 5; i++) { console.log(`${{i}}回目`); } // → 0回目、1回目、2回目、3回目、4回目 // 1〜10を合計する let total = 0; for (let i = 1; i <= 10; i++) { total += i; } console.log(`1〜10の合計：${{total}}`); // 55 // 逆順ループ for (let i = 5; i >= 1; i--) { console.log(i); // 5, 4, 3, 2, 1 }
script.js（while文）
// while（条件がtrueの間繰り返す） let count = 0; while (count < 5) { console.log(`count: ${{count}}`); count++; // ← 忘れると無限ループ！ } // do...while（必ず1回は実行される） let num = 10; do { console.log(num); // 条件に関係なく1回は実行される num++; } while (num < 5); // falseなので1回で終了
forとwhileの使い分け
繰り返す回数が分かるかどうかで決める
for文：繰り返す回数がはっきり決まっているとき
例：配列の全要素を処理する、10回繰り返す

while文：条件が満たされるまで繰り返すとき
例：ユーザーが正しい入力をするまで繰り返す

実務ではfor文が圧倒的に多く使われます。
2
break・continue・forOfを使おう
25〜40分
📖
breakでループを抜け出し、continueで特定の回をスキップできます。配列にはfor...ofが便利です。
script.js（break・continue）
// break：ループを途中で終了する for (let i = 0; i < 10; i++) { if (i === 5) break; // i が5になったら抜ける console.log(i); // 0, 1, 2, 3, 4 } // continue：今回の繰り返しをスキップ for (let i = 0; i < 10; i++) { if (i % 2 === 0) continue; // 偶数はスキップ console.log(i); // 1, 3, 5, 7, 9（奇数だけ） }
script.js（for...of で配列をループ）
const fruits = ["りんご", "バナナ", "みかん", "ぶどう"]; // for文でインデックスを使う方法 for (let i = 0; i < fruits.length; i++) { console.log(fruits[i]); } // for...of：シンプルで読みやすい（配列の要素を順番に取り出す） for (const fruit of fruits) { console.log(fruit); // りんご、バナナ、みかん、ぶどう } // インデックスも必要なら entries() を使う for (const [i, fruit] of fruits.entries()) { console.log(`${{i + 1}}番目：${{fruit}}`); }
💡
for...ofはES2015（ES6）から使える書き方です。配列を順番に処理するだけなら for...of が最もシンプルで読みやすいです。インデックスが必要な場合は entries() を組み合わせましょう。
