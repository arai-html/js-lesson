演算子を使いこなそう
0〜20分
📖
演算子は値を計算・比較するための記号です。条件分岐で使う比較演算子と論理演算子が特に重要です。
script.js（算術演算子）
const a = 10; const b = 3; console.log(a + b); // 13 足し算 console.log(a - b); // 7 引き算 console.log(a * b); // 30 掛け算 console.log(a / b); // 3.333... 割り算 console.log(a % b); // 1 余り（モジュロ） console.log(a ** b); // 1000 べき乗（10の3乗） // 代入演算子（短縮形） let count = 0; count += 1; // count = count + 1 と同じ → 1 count *= 3; // count = count * 3 と同じ → 3 count++; // count = count + 1 と同じ → 4（インクリメント） count--; // count = count - 1 と同じ → 3（デクリメント）
script.js（比較・論理演算子）
// 比較演算子（結果はtrue/false） console.log(10 === 10); // true 完全一致（型も比較） console.log(10 === "10"); // false 型が違うのでfalse console.log(10 !== 5); // true 不一致 console.log(10 > 5); // true より大きい console.log(10 >= 10); // true 以上 console.log(10 < 5); // false より小さい // 論理演算子 console.log(true && true); // true AND：両方trueのとき console.log(true || false); // true OR：どちらかtrueのとき console.log(!true); // false NOT：反転
== と === の違い
現代のJSでは=== を使う
==（二重イコール）は型を自動変換してから比較します。10 == "10" が true になるなど、予期しない結果になることがあります。

===（三重イコール）は型も含めて厳密に比較します。現代のJavaScriptでは常に=== を使うのが基本です。

条件分岐（if/else・三項演算子）
20〜40分
📖
if/else文で条件によって処理を分岐させます。シンプルな分岐は三項演算子でも書けます。
script.js（if/else）
const score = 75; // 基本のif/else if (score >= 80) { console.log("合格！"); } else if (score >= 60) { console.log("もう少し！"); // ← これが実行される } else { console.log("不合格"); } // 複数条件の組み合わせ（&&・||） const age = 20; const hasTicket = true; if (age >= 18 && hasTicket) { console.log("入場できます"); } else { console.log("入場できません"); }
script.js（三項演算子）
// 三項演算子：条件 ? trueのとき : falseのとき const score = 75; // if/elseで書いた場合 let result1; if (score >= 60) { result1 = "合格"; } else { result1 = "不合格"; } // 三項演算子で書いた場合（1行でスッキリ！） const result2 = score >= 60 ? "合格" : "不合格"; console.log(result1); // "合格" console.log(result2); // "合格" // テンプレートリテラルと組み合わせる console.log(`判定：${{score >= 60 ? "合格" : "不合格"}}`);
💡
三項演算子は読みやすさが命です。条件が複雑になるときはif/elseで書く方が親切。1行で書ける単純な分岐のときだけ三項演算子を使うのがベストプラクティスです。
