正規表現の基本を覚えよう
0〜25分
💬
講師がこう話す「正規表現は文字列のパターンを表す記法です。メールアドレスの形式チェック・電話番号の抽出・テキストの置換など、文字列操作の強力なツールです。最初は記号だらけで怖いですが、よく使うパターンを5〜6個覚えれば実務の8割はこなせます。」
script.js（正規表現の基本）
// 正規表現リテラル：/パターン/フラグ const pattern = /hello/; // "hello"にマッチ const patternI = /hello/i; // 大文字小文字を無視（フラグi） const patternG = /hello/g; // 全てにマッチ（フラグg） // test()：パターンにマッチするか true/false で返す console.log(/hello/.test("hello world")); // true console.log(/hello/.test("HELLO")); // false console.log(/hello/i.test("HELLO")); // true（iフラグで大文字無視） // match()：マッチした部分を配列で返す const str = "2026年3月13日"; console.log(str.match(/\d+/g)); // ["2026", "3", "13"] // replace()：マッチした部分を置換 const text = " hello world "; console.log(text.replace(/\s+/g, " ").trim()); // "hello world"
script.js（よく使うパターン）
// \d 数字1文字　\D 数字以外 // \w 英数字+_　 \W それ以外 // \s 空白文字 \S 空白以外 // . 任意の1文字（改行以外） // ^ 文字列の先頭　$ 文字列の末尾 // * 0回以上　+ 1回以上　? 0か1回 // {n} n回　{n,m} n〜m回 // [] 文字クラス（例：[a-z] 小文字英字） // 実践例 const emailReg = /^[\w.-]+@[\w.-]+\.\w{2,}$/; const telReg = /^\d{2,4}-\d{2,4}-\d{4}$/; const zipReg = /^\d{3}-\d{4}$/; console.log(emailReg.test("test@example.com")); // true console.log(emailReg.test("not-an-email")); // false console.log(telReg.test("090-1234-5678")); // true console.log(zipReg.test("150-0001")); // true
💡
正規表現は最初から全部覚えようとしなくて大丈夫です。よく使う5パターン（メール・電話番号・郵便番号・数字のみ・英数字のみ）をコピーして使えるようにしておけば実務は十分こなせます。必要なときにMDNやregex101で調べる習慣をつけましょう。
👆
講師メモregex101.comは正規表現のデバッグに便利。実際に開いて生徒に使い方を紹介すると良い。「正規表現を暗記する必要はない、理解して検索できればOK」と伝えてハードルを下げる。

フォームバリデーションに応用しよう
25〜45分
💬
講師がこう話す「正規表現の最も実践的な使い方がフォームバリデーションです。メールアドレス・電話番号・パスワードの形式チェックを、入力のたびにリアルタイムで行うUIを作ります。」
script.js（リアルタイムバリデーション）
// バリデーションルールを定義 const rules = { name: { reg: /^.{2,20}$/, msg: "2〜20文字で入力してください" }, email: { reg: /^[\w.-]+@[\w.-]+\.\w{2,}$/, msg: "正しいメールアドレスを入力してください"}, tel: { reg: /^\d{10,11}$/, msg: "10〜11桁の数字で入力してください" }, password: { reg: /^(?=.*[A-Z])(?=.*\d).{8,}$/, msg: "大文字・数字を含む8文字以上" } }; // 各inputにバリデーションを適用 Object.keys(rules).forEach(fieldName => { const input = document.querySelector(`#${{fieldName}}`); const error = document.querySelector(`#${{fieldName}}-error`); if (!input) return; input.addEventListener("blur", () => { const { reg, msg } = rules[fieldName]; const isValid = reg.test(input.value); input.classList.toggle("error", !isValid); input.classList.toggle("success", isValid); if (error) error.textContent = isValid ? "" : msg; }); }); // 送信前に全フィールドを一括チェック const validateAll = () => { return Object.keys(rules).every(fieldName => { const input = document.querySelector(`#${{fieldName}}`); return input ? rules[fieldName].reg.test(input.value) : true; }); };
