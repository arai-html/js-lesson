イベントを登録しよう
0〜25分
📖
addEventListenerでイベントを登録し、ユーザーの操作に応じた処理を実行します。
script.js（addEventListenerの基本）
const btn = document.querySelector("#btn"); // addEventListener(イベント名, 処理する関数) btn.addEventListener("click", () => { console.log("クリックされました！"); }); // 関数を別に定義して渡す方法（再利用できる） const handleClick = () => { console.log("ハンドラー関数が実行された"); }; btn.addEventListener("click", handleClick); // イベントの削除（不要になったら解除する） btn.removeEventListener("click", handleClick); // よく使うイベント一覧 // click クリック // dblclick ダブルクリック // mouseenter マウスが要素に入ったとき // mouseleave マウスが要素から出たとき // keydown キーを押したとき // keyup キーを離したとき // input 入力のたびに発火 // change 値が変わってフォーカスが外れたとき // submit フォームが送信されたとき // scroll スクロールしたとき
script.js（イベントオブジェクト）
// イベント処理の関数はイベントオブジェクト(e)を受け取れる const btn = document.querySelector("#btn"); btn.addEventListener("click", (e) => { console.log(e.type); // "click"（イベントの種類） console.log(e.target); // クリックされた要素 console.log(e.target.id); // 要素のid console.log(e.clientX, e.clientY); // クリック位置（座標） }); // フォームの送信をキャンセルする（最重要！） const form = document.querySelector("form"); form.addEventListener("submit", (e) => { e.preventDefault(); // ページリロードをキャンセル console.log("フォームが送信されました"); });
preventDefault()は必須
フォームとリンクで特に重要
e.preventDefault()を呼ぶと、ブラウザのデフォルト動作をキャンセルできます。

• form の submit：ページがリロードされるのを防ぐ
• a タグのクリック：ページ遷移を防ぐ（スムーズスクロールなどに使う）

これを忘れるとフォームを送信するたびにページがリロードされます。

イベントデリゲーション・フォームイベント
25〜40分
📖
イベントデリゲーションは親要素でイベントをまとめて受け取るテクニックです。動的に追加した要素にも有効です。
script.js（イベントデリゲーション）
// ❌ 個々の要素にイベントを登録（後から追加した要素には効かない） document.querySelectorAll(".item").forEach(item => { item.addEventListener("click", () => console.log("クリック")); }); // ✅ 親要素にイベントを登録（動的追加後も有効） const list = document.querySelector("#list"); list.addEventListener("click", (e) => { // クリックされたのが .item のときだけ処理 if (e.target.classList.contains("item")) { console.log(`${{e.target.textContent}}がクリックされた`); e.target.classList.toggle("selected"); } });
script.js（フォームイベント）
const input = document.querySelector("#name-input"); const preview = document.querySelector("#preview"); // input：入力するたびに発火（リアルタイム） input.addEventListener("input", (e) => { preview.textContent = e.target.value; }); // focus：フォーカスが当たったとき input.addEventListener("focus", () => { input.style.borderColor = "#f7df1e"; }); // blur：フォーカスが外れたとき（バリデーションに使う） input.addEventListener("blur", () => { input.style.borderColor = ""; if (input.value.trim() === "") { input.classList.add("error"); } else { input.classList.remove("error"); } });
💡
inputイベントはchangeイベントと違い、入力のたびにリアルタイムで発火します。文字数カウンター・リアルタイム検索・プレビュー表示など、入力に即座に反応するUIを作るときに使います。
