1
JavaScriptでできること
0〜10分
💬
講師がこう話す「HTMLは骨格、CSSはデザイン、JavaScriptは動きです。ボタンを押したらメニューが開く・スクロールしたらアニメーションする・フォームの入力チェックをする——これら全部JavaScriptの仕事です。」
🌐
JavaScriptが使われる場面
• ボタンをクリックしたらメニューが開く
• フォームの入力内容をチェックする
• 画像がスライドする・要素がふわっと現れる
• サーバーからデータを取得して画面に表示する
• ゲーム・地図・SNSのリアルタイム更新
HTML・CSS・JSの役割分担
3つの言語は互いに補い合っている
HTML：ページの骨格・構造（何があるか）
CSS：見た目・デザイン（どう見えるか）
JavaScript：動き・インタラクション（どう動くか）

JavaScriptはHTMLの要素を取得して、中身を書き換えたりスタイルを変えたりします。
👆
講師メモ生徒が毎日使っているWebサービス（SNS・ショッピングサイト・動画サービス）を例に出すと身近に感じる。「あの動きはJSで作られている」と伝えるとモチベーションが上がる。
2
scriptタグでJSを書こう
10〜25分
💬
講師がこう話す「JavaScriptはHTMLファイルにscriptタグを書くだけで動きます。インストール不要・ブラウザだけで実行できます。scriptタグはbodyの閉じタグの直前に書くのが基本です。」
index.html
<!DOCTYPE html> <html lang="ja"> <head> <meta charset="UTF-8"> <title>はじめてのJS</title> </head> <body> <h1 id="message">こんにちは！</h1> <button id="btn">クリックしてね</button> <!-- bodyの閉じタグ直前にscriptタグを書く --> <script src="script.js"></script> </body> </html>
script.js
// console.log：開発者ツールのConsoleに出力 console.log("Hello, JavaScript!"); // alert：ポップアップを表示 alert("ページが読み込まれました！"); // ボタンをクリックしたらh1のテキストを変える const btn = document.querySelector("#btn"); btn.addEventListener("click", function() { document.querySelector("#message").textContent = "ボタンが押されました！"; });
💡
F12キー（またはCmd+Option+I）でデベロッパーツールを開き、Consoleタブを見てみましょう。console.log()の出力がここに表示されます。エラーもここに表示されるので、デバッグの第一歩です。
👆
講師メモF12→Consoleタブを先に開いておく。console.logが出たときの達成感を大切にする。alertは次に確認させる。
3
prompt・型・コメントを覚えよう
25〜40分
💬
講師がこう話す「promptはユーザーに入力を求めるダイアログを表示します。入力した値を変数に受け取って使うことができます。JavaScriptには文字列・数値・真偽値など複数のデータ型があります。」
script.js
// prompt：入力ダイアログを表示して値を受け取る const name = prompt("あなたの名前を入力してください"); alert("こんにちは、" + name + "さん！"); // console.logで確認 console.log("名前：", name); console.log("型：", typeof name); // "string"と表示される // コメントの書き方 // 1行コメント：//の後に書く /* 複数行コメント： この間は全部コメントになる */
JavaScriptの主なデータ型
typeof演算子で確認できる
string　→ 文字列　例：'こんにちは'
number　→ 数値　例：42、3.14
boolean → 真偽値　例：true、false
undefined → 値が未定義
null　→ 意図的に「空」を表す
