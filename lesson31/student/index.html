なぜReactが必要なのか
0〜15分
📖
Reactはデータが変わるとUIを自動で再描画してくれるUIライブラリです。大規模アプリの状態管理を劇的にシンプルにします。
⚛️
バニラJSとReactの違い
バニラJS：
① データを変更する
② render()を手動で呼ぶ
③ DOMを自分で更新する

React：
① stateを変更する（setState）
② Reactが自動でUIを再描画
③ 変更があった部分だけ効率よく更新（Virtual DOM）

「状態を変えればUIは自動でついてくる」がReactの約束です。
index.html（Reactをscriptタグで読み込む）
<!-- CDNでReactを読み込む（学習用） --> <script src="https://unpkg.com/react@18/umd/react.development.js"></script> <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script> <script src="https://unpkg.com/@babel/standalone/babel.min.js"></script> <div id="root"></div> <!-- type="text/babel" でJSXをブラウザで変換 --> <script type="text/babel"> // Reactコードをここに書く </script>

JSXとコンポーネントの基本
15〜40分
📖
JSXはJSの中にHTML風の記法でUIを書ける構文です。コンポーネントはJSXを返す関数です。
app.jsx（JSXの基本）
// JSX：JSの中にHTMLのような記法でUIを書く const Hello = () => { return ( <div className="hello"> <h1>こんにちは！</h1> <p>Reactへようこそ</p> </div> ); }; // JSXのルール // ① classではなくclassName // ② 複数要素は1つの要素で囲む（または<>...） // ③ {}でJSの式を埋め込める // ④ self-closingタグは<br />のように閉じる const name = "田中"; const Greeting = () => ( <> <h1>こんにちは、{name}さん！</h1> <p>今日は{new Date().toLocaleDateString("ja-JP")}です</p> <p>{1 + 2 === 3 ? "計算OK" : "バグ！"}</p> </> ); // ReactDOMでrootにマウント const root = ReactDOM.createRoot(document.getElementById("root")); root.render(<Greeting />);
app.jsx（コンポーネントとprops）
// propsは親から子へデータを渡す仕組み const UserCard = ({ name, age, role }) => ( <div className="user-card"> <h3>{name}</h3> <p>{age}歳 / {role}</p> </div> ); // 配列からコンポーネントを動的に生成（keyが必要！） const users = [ { id: 1, name: "田中", age: 25, role: "エンジニア" }, { id: 2, name: "山田", age: 30, role: "デザイナー" }, { id: 3, name: "鈴木", age: 22, role: "マーケター" }, ]; const UserList = () => ( <div className="user-list"> <h2>メンバー一覧</h2> {users.map(user => ( <UserCard key={user.id} name={user.name} age={user.age} role={user.role} /> ))} </div> ); root.render(<UserList />);
keyプロパティはなぜ必要？
Reactがリストの差分を効率よく検出するため
リストをmap()でレンダリングするとき、各要素にkeyプロパティが必要です。

Reactはkeyを使ってどの要素が変わったかを判断し、変更のある部分だけ再描画します（Virtual DOM）。

• ✅ keyはユニークな値（id・indexは最終手段）
• ❌ keyを省略すると警告が出てパフォーマンスが落ちる
