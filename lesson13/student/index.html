ローカルストレージを使おう
0〜25分
📖
ローカルストレージはブラウザにデータを永続保存できる仕組みです。リロードしても消えません。
💾
WebStorageの種類
localStorage：ブラウザを閉じても保存が続く。容量は約5MB。
sessionStorage：タブを閉じると消える。タブ間でデータを共有しない。

今回はlocalStorageを使います。キーと値のペアで保存し、値は文字列のみです。
script.js（localStorageの基本操作）
// 保存（setItem） localStorage.setItem("username", "田中太郎"); localStorage.setItem("theme", "dark"); localStorage.setItem("count", "42"); // 数値も文字列で保存 // 取得（getItem）← 存在しない場合はnullが返る const name = localStorage.getItem("username"); const theme = localStorage.getItem("theme"); console.log(name, theme); // "田中太郎" "dark" // 数値として使うには変換が必要 const count = Number(localStorage.getItem("count")); console.log(count + 1); // 43 // 削除（removeItem） localStorage.removeItem("theme"); // 全削除（clear） localStorage.clear(); // キー数を確認 console.log(localStorage.length);
値は必ず文字列
数値・真偽値は変換が必要
localStorageは値をすべて文字列として保存します。

• 数値を保存 → String(42) または '42'
• 取り出して使う → Number(getItem('count'))
• 真偽値 → 'true'/'false'で保存し取り出し時に比較

オブジェクト・配列は後述のJSON.stringifyを使います。
2
JSONと組み合わせてオブジェクトを保存しよう
25〜45分
📖
JSON.stringify/parseを使うことで、配列やオブジェクトもlocalStorageに保存できます。
script.js（オブジェクト・配列の保存）
// オブジェクトを保存する const user = { name: "太郎", age: 25, premium: true }; // JSON.stringify：オブジェクト → 文字列に変換 localStorage.setItem("user", JSON.stringify(user)); // JSON.parse：文字列 → オブジェクトに戻す const saved = JSON.parse(localStorage.getItem("user")); console.log(saved.name); // "太郎" console.log(saved.premium); // true（真偽値として復元される） // nullチェック（初回アクセス時はnullになる） const raw = localStorage.getItem("user"); const userData = raw ? JSON.parse(raw) : null;
script.js（ToDoをlocalStorageに保存）
// 前回作ったToDoリストにlocalStorage保存を追加 // 初期データ：保存されていればそれを使い、なければ空配列 let todos = JSON.parse(localStorage.getItem("todos")) || []; // 保存する関数 const saveTodos = () => { localStorage.setItem("todos", JSON.stringify(todos)); }; // タスク追加のたびに保存 const addTodo = (text) => { todos.push({ text, done: false }); saveTodos(); // ← 追加後に保存 render(); }; // 削除のたびにも保存 const deleteTodo = (i) => { todos.splice(i, 1); saveTodos(); // ← 削除後に保存 render(); };
💡
JSON.parse(localStorage.getItem('key')) || [] というパターンは必須で覚えましょう。初回（データが存在しない）はnullが返るので、|| [] でフォールバックします。|| {{}} でオブジェクトのフォールバックも同様です。
