📖
クイズアプリはタイマー・シャッフル・状態管理・DOM切り替えを使う総合演習です。
🧠
クイズアプリの完成機能
① 問題データ（配列）からランダムに出題
② 4択から選択 → 正解/不正解を色で表示
③ 1問につき15秒のカウントダウンタイマー
④ タイムアップで自動的に次の問題へ
⑤ 全問終了で結果画面（スコア・正答率）
⑥ 「もう一度」でリセット
script.js（問題データとシャッフル）
// 問題データ（実際には多く用意する） const questions = [ { question: "JavaScriptで配列の最後に要素を追加するメソッドは？", choices: ["push", "pop", "shift", "unshift"], answer: 0 // 正解のインデックス }, { question: "NaNを検出するJavaScriptの関数は？", choices: ["isNull()", "isUndefined()", "isNaN()", "isEmpty()"], answer: 2 }, { question: "Promiseの状態として正しくないものは？", choices: ["pending", "fulfilled", "rejected", "waiting"], answer: 3 }, { question: "CSSでフレックスコンテナを作るプロパティは？", choices: ["display: grid", "display: flex", "float: left", "position: flex"], answer: 1 }, ]; // 配列をシャッフルする（Fisher-Yatesアルゴリズム） const shuffle = (arr) => { const a = [...arr]; for (let i = a.length - 1; i > 0; i--) { const j = Math.floor(Math.random() * (i + 1)); [a[i], a[j]] = [a[j], a[i]]; // 分割代入でスワップ } return a; };
script.js（状態管理・メイン処理）
// 状態 let state = { questions: [], // シャッフル済み問題 current: 0, // 現在の問題インデックス score: 0, // 正解数 answered: false, // この問題に回答済みか timeLeft: 15, // 残り時間 timerId: null, // setIntervalのID }; // ゲームを開始 const startGame = () => { state = { questions: shuffle(questions), current: 0, score: 0, answered: false, timeLeft: 15, timerId: null, }; renderQuestion(); startTimer(); }; // 回答処理 const answer = (index) => { if (state.answered) return; // 回答済みは無視 state.answered = true; clearInterval(state.timerId); // タイマー停止 const q = state.questions[state.current]; const correct = index === q.answer; if (correct) state.score++; showAnswer(index, q.answer, correct); // 1秒後に次の問題へ setTimeout(() => nextQuestion(), 1500); };

タイマー・結果画面・CSS仕上げ
25〜45分
📖
setIntervalでタイマーを実装し、全問終了後の結果画面と「もう一度」ボタンを実装します。
script.js（タイマー・次の問題・結果画面）
// タイマー処理 const startTimer = () => { state.timeLeft = 15; updateTimerUI(); state.timerId = setInterval(() => { state.timeLeft--; updateTimerUI(); if (state.timeLeft <= 0) { clearInterval(state.timerId); if (!state.answered) answer(-1); // 時間切れ→不正解扱い } }, 1000); }; const updateTimerUI = () => { const el = document.querySelector("#timer"); if (!el) return; el.textContent = state.timeLeft; el.classList.toggle("danger", state.timeLeft <= 5); // 残り5秒で赤色 }; // 次の問題へ const nextQuestion = () => { state.current++; if (state.current >= state.questions.length) { showResult(); // 全問終了 return; } state.answered = false; state.timeLeft = 15; renderQuestion(); startTimer(); }; // 結果画面 const showResult = () => { const total = state.questions.length; const pct = Math.round(state.score / total * 100); const message = pct >= 80 ? "🎉 素晴らしい！" : pct >= 50 ? "👍 もう少し！" : "📚 もっと勉強しよう！"; document.querySelector("#quiz-area").innerHTML = ` <div class="result"> <h2>結果発表</h2> <div class="score">${{state.score}} / ${{total}}</div> <div class="percent">正答率 ${{pct}}%</div> <p class="message">${{message}}</p> <button id="retry-btn">もう一度挑戦！</button> </div> `; document.querySelector("#retry-btn").addEventListener("click", startGame); }; startGame(); // ゲーム開始
💡
タイマーは必ずclearInterval(state.timerId)で止めてから次に進みましょう。止め忘れると古いタイマーと新しいタイマーが重複して動き、予期しない動作の原因になります。「タイマーを使ったら必ず止める」を鉄則にしましょう。
