DOMを取得して中身を変えよう
0〜25分
📖
DOMはHTMLをJavaScriptから操作するための仕組みです。要素を取得して中身やスタイルを変えられます。
🌐
DOMで何ができる？
• テキストや画像を書き換える
• クラスを追加・削除してスタイルを切り替える
• 新しい要素を作って追加する
• 要素を削除する
• イベント（クリック・入力など）を検知して処理する
script.js（要素の取得）
// querySelector：CSSセレクタで1つ取得 const title = document.querySelector("h1"); const btn = document.querySelector("#submit-btn"); const card = document.querySelector(".card"); // querySelectorAll：条件に一致する全要素をNodeListで取得 const items = document.querySelectorAll(".item"); const links = document.querySelectorAll("a"); // 取得できたか確認（nullチェックは必須！） console.log(btn); // 要素オブジェクト or null console.log(items.length); // 取得できた要素数 // NodeListにはforEachが使える items.forEach(item => { console.log(item.textContent); });
script.js（テキスト・HTML・属性の変更）
const title = document.querySelector("#title"); const link = document.querySelector("a"); const img = document.querySelector("img"); // テキストを変える（HTMLタグはそのまま文字列になる・安全） title.textContent = "新しいタイトル"; // HTMLごと変える（タグが有効になる・XSSに注意） title.innerHTML = "<span>タイトル</span>"; // 属性を取得・変更 console.log(link.getAttribute("href")); // 現在のhref link.setAttribute("href", "https://example.com"); link.setAttribute("target", "_blank"); // src・alt などはプロパティで直接変更できる img.src = "new-image.jpg"; img.alt = "新しい画像";
textContent vs innerHTML
使い分けが重要
textContent：テキストとして扱う。HTMLタグも文字列になる。ユーザー入力を表示するときは必ずこちら（XSS対策）

innerHTML：HTMLとして解釈する。タグが有効になる。自分で書いた静的なHTMLを挿入するときに使う。

クラス操作・スタイル変更
25〜40分
📖
CSSのクラスをJSで追加・削除してスタイルを切り替えるのが実務の基本パターンです。
script.js（クラス操作）
const btn = document.querySelector("#btn"); const box = document.querySelector(".box"); // クラスを追加・削除・トグル・確認 box.classList.add("active"); // 追加 box.classList.remove("active"); // 削除 box.classList.toggle("active"); // あれば削除・なければ追加 box.classList.contains("active"); // 持っているか → true/false // ボタンクリックでトグル（実務の基本パターン） btn.addEventListener("click", () => { box.classList.toggle("open"); btn.textContent = box.classList.contains("open") ? "閉じる" : "開く"; });
script.js（スタイルを直接変更）
const box = document.querySelector(".box"); // style プロパティで直接変更（CSSプロパティはキャメルケース） box.style.backgroundColor = "#f7df1e"; // background-color → backgroundColor box.style.fontSize = "20px"; box.style.display = "none"; // 非表示 box.style.display = ""; // 元に戻す（スタイルを削除） // 現在のスタイルを取得（getComputedStyleを使う） const computed = getComputedStyle(box); console.log(computed.backgroundColor); console.log(computed.width);
💡
スタイルを直接変えるよりクラスを切り替える方が推奨されます。理由はデザインの管理をCSSに任せられるため。JSはクラスを付けるだけ、見た目の詳細はCSSで定義——この役割分担が保守しやすいコードを作ります。
