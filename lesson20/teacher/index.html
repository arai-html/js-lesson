フォームバリデーションを実装しよう
0〜25分
💬
講師がこう話す「フォームバリデーションはユーザーに正しいデータを入力してもらうためのUX改善の最前線です。HTML5の標準バリデーションとJavaScriptによるカスタムバリデーションを組み合わせて、使いやすいフォームを作りましょう。」
index.html（HTML5バリデーション属性）
<form id="signup-form" novalidate> <!-- required：必須入力 --> <input type="text" id="name" required minlength="2" maxlength="20"> <!-- type="email"：メール形式チェック --> <input type="email" id="email" required> <!-- pattern：正規表現でチェック --> <input type="tel" id="tel" pattern="[0-9]{10,11}"> <!-- min/max：数値の範囲 --> <input type="number" id="age" min="18" max="100"> <button type="submit">登録</button> </form>
script.js（Constraint Validation API）
// novalidateでブラウザデフォルトのUIを無効化してJSで制御 const form = document.querySelector("#signup-form"); const showError = (input, message) => { input.classList.add("is-invalid"); input.classList.remove("is-valid"); const errEl = input.nextElementSibling; if (errEl?.classList.contains("error-msg")) errEl.textContent = message; }; const showSuccess = (input) => { input.classList.remove("is-invalid"); input.classList.add("is-valid"); const errEl = input.nextElementSibling; if (errEl?.classList.contains("error-msg")) errEl.textContent = ""; }; // HTML5のConstraint Validation APIを使ってエラー内容を確認 const validate = (input) => { if (input.validity.valueMissing) { showError(input, "必須項目です"); } else if (input.validity.tooShort) { showError(input, `最低${{input.minLength}}文字必要です`); } else if (input.validity.typeMismatch) { showError(input, "正しい形式で入力してください"); } else if (input.validity.patternMismatch) { showError(input, "形式が正しくありません"); } else { showSuccess(input); } };
validity オブジェクトの主なプロパティ
HTML5バリデーションのエラー種別を判別できる
valueMissing：required なのに空
tooShort/tooLong：minlength/maxlength違反
typeMismatch：type=emailなのにメール形式でない
patternMismatch：pattern正規表現に不一致
rangeUnderflow/rangeOverflow：min/max違反
valid：全て問題なし（true/false）
👆
講師メモnovalidate属性をformに付けてブラウザ標準UIを無効にし、JSで独自UIを実装するのがプロの作り方。ブラウザごとにデフォルトUIが異なるため、統一感のあるデザインにするために必要。

"非常に弱い"; }; const pwInput = document.querySelector("#password"); const strength = document.querySelector("#strength"); pwInput.addEventListener("input", () => { strength.textContent = getStrength(pwInput.value); }); // パスワード確認の一致チェック const pwConfirm = document.querySelector("#password-confirm"); pwConfirm.addEventListener("input", () => { const match = pwInput.value === pwConfirm.value; pwConfirm.classList.toggle("is-valid", match); pwConfirm.classList.toggle("is-invalid", !match); });
