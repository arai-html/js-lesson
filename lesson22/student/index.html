データからUIを動的に生成しよう
0〜25分
📖
配列データをmap()でHTMLに変換して描画するパターンは実務の基本です。フィルター・ソートも合わせて実装します。
script.js（データ→カードUI生成）
// 商品データ const products = [ { id: 1, name: "ノートPC", price: 98000, category: "PC", rating: 4.5 }, { id: 2, name: "マウス", price: 3800, category: "周辺機器", rating: 4.2 }, { id: 3, name: "キーボード", price: 8500, category: "周辺機器", rating: 4.7 }, { id: 4, name: "モニター", price: 42000, category: "PC", rating: 4.3 }, { id: 5, name: "Webカメラ", price: 6800, category: "周辺機器", rating: 3.9 }, ]; // カードHTMLを生成する関数 const createCard = (product) => ` <div class="card" data-id="${{product.id}}"> <div class="card-category">${{product.category}}</div> <h3 class="card-name">${{product.name}}</h3> <div class="card-price">¥${{product.price.toLocaleString()}}</div> <div class="card-rating">★ ${{product.rating}}</div> <button class="add-to-cart" data-id="${{product.id}}"> カートに追加 </button> </div> `; // 一覧を描画する関数 const container = document.querySelector("#product-list"); const render = (items) => { container.innerHTML = items.map(createCard).join(""); }; render(products); // 初期表示
描画の設計パターン
データと表示を分離する
render(items) という「データを受け取って画面を更新する関数」を1つ定義しておき、フィルター・ソート後に必ずこれを呼ぶ設計が保守しやすいコードの基本です。

① 元データ（products）は変更しない
② フィルター・ソートした結果を別変数に入れる
③ render()に渡す

この3ステップを守ると、機能追加が格段に楽になります。

フィルター・ソート・ページネーション
25〜45分
📖
フィルター・ソート・ページネーションを状態管理と組み合わせて実装します。
script.js（フィルター・ソート・ページネーション）
// 状態管理 let state = { category: "all", sortBy: "default", currentPage: 1, perPage: 3, }; // フィルター＋ソートした結果を返す const getFiltered = () => { let result = [...products]; // 元データをコピー（破壊しない） // フィルター if (state.category !== "all") { result = result.filter(p => p.category === state.category); } // ソート if (state.sortBy === "price-asc") result.sort((a, b) => a.price - b.price); if (state.sortBy === "price-desc") result.sort((a, b) => b.price - a.price); if (state.sortBy === "rating-desc") result.sort((a, b) => b.rating - a.rating); return result; }; // ページネーション付きで描画 const renderWithPage = () => { const filtered = getFiltered(); const start = (state.currentPage - 1) * state.perPage; const paged = filtered.slice(start, start + state.perPage); render(paged); renderPagination(filtered.length); }; // ページネーションを描画 const renderPagination = (total) => { const pages = Math.ceil(total / state.perPage); const pageEl = document.querySelector("#pagination"); pageEl.innerHTML = Array.from({ length: pages }, (_, i) => ` <button class="page-btn ${{i + 1 === state.currentPage ? "active" : ""}}" data-page="${{i + 1}}">${{i + 1}}</button> `).join(""); }; // カテゴリフィルターのイベント document.querySelector("#category-filter")?.addEventListener("change", (e) => { state.category = e.target.value; state.currentPage = 1; // フィルター時はページを1に戻す renderWithPage(); }); renderWithPage(); // 初期描画
💡
フィルター・ソートするときは必ず元の配列をスプレッド構文でコピーしてから操作しましょう（[...products]）。sort()は元の配列を変更する破壊的メソッドなので、コピーしないと元データが失われます。
