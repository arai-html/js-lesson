jQueryとバニラJSの関係
jQueryは2006年に登場し、ブラウザ間の非互換を吸収する目的で生まれました。

現在はブラウザが標準化されたため、新規開発でjQueryを使う必要性は薄れています。

しかし既存のWordPressサイト・保守案件では現役です。「読める・書ける」は実務で役立ちます。
比較（DOM操作）
// ── 要素の取得 ── // jQuery const btn = $("#submit-btn"); const items = $(".item"); // バニラJS const btn2 = document.querySelector("#submit-btn"); const items2 = document.querySelectorAll(".item"); // ── テキスト・HTML変更 ── $("#title").text("新タイトル"); // jQuery document.querySelector("#title").textContent = "新タイトル"; // JS // ── クラス操作 ── $(".box").addClass("active"); // jQuery document.querySelector(".box").classList.add("active"); // JS $(".box").toggleClass("open"); // jQuery document.querySelector(".box").classList.toggle("open"); // JS // ── イベント ── $("#btn").on("click", () => console.log("click")); // jQuery document.querySelector("#btn") // JS .addEventListener("click", () => console.log("click")); // ── Ajax（非同期通信）── // jQuery（$.ajax / $.get） $.get("https://jsonplaceholder.typicode.com/posts/1", (data) => { console.log(data.title); }); // バニラJS（fetch） fetch("https://jsonplaceholder.typicode.com/posts/1") .then(r => r.json()) .then(data => console.log(data.title));
jQueryが今でも使われる場面
使い分けを知ろう
jQueryが活躍する場面：
• WordPressのテーマ・プラグイン開発
• 既存サイトの保守・機能追加

新規開発ではバニラJS・Reactを使う：

jQueryを読む・書く練習
25〜45分
📖
jQueryの書き方に慣れておきましょう。バニラJSを知っていれば難しくありません。
jQuery（アコーディオン・タブの実装例）
// ── jQueryでアコーディオン ── $(".accordion-header").on("click", function() { $(this).next(".accordion-body").slideToggle(300); $(this).toggleClass("open"); }); // ── バニラJSで同じことをする ── document.querySelectorAll(".accordion-header").forEach(header => { header.addEventListener("click", () => { const body = header.nextElementSibling; header.classList.toggle("open"); // slideToggleはCSSで再現 body.style.maxHeight = header.classList.contains("open") ? body.scrollHeight + "px" : "0"; }); }); // ── jQueryでシンプルなタブ ── $(".tab-btn").on("click", function() { const target = $(this).data("target"); $(".tab-btn").removeClass("active"); $(".tab-pane").hide(); $(this).addClass("active"); $(`#${{target}}`).show(); });
💡
jQueryの$(this)はクリックされた要素自身を指します。バニラJSのイベントハンドラ内のe.currentTargetやe.targetに相当します。jQueryではアロー関数の中ではthisが使えないのでfunctionを使います。
