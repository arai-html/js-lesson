天気アプリの完成イメージ
機能：
① 都市名を入力して検索ボタンを押す
② OpenWeatherMap APIにfetchでリクエスト
③ 天気・気温（摂氏）・湿度・天気アイコンを表示
④ ローディング中はスピナー表示
⑤ 都市が見つからない・通信エラーのときはエラーメッセージ表示
⑥ 最後に検索した都市をlocalStorageに保存して次回復元
index.html（HTML骨格）
<div class="weather-app"> <h1>🌤 天気アプリ</h1> <!-- 検索フォーム --> <div class="search-box"> <input type="text" id="city-input" placeholder="都市名を英語で入力 例: Tokyo"> <button id="search-btn">検索</button> </div> <!-- ローディング --> <div id="loading" class="loading" hidden> <div class="spinner"></div> <p>取得中...</p> </div> <!-- エラーメッセージ --> <div id="error-msg" class="error" hidden></div> <!-- 天気表示エリア --> <div id="weather-result" hidden> <h2 id="city-name"></h2> <img id="weather-icon" alt="天気アイコン"> <p id="weather-desc"></p> <div id="weather-details"></div> </div> </div>

講師メモOpenWeatherMapのAPIキーは事前に取得してもらう。無料プランで十分。取得→サインアップ→API keys→生成→数分〜数時間で有効化される。当日使えない可能性があるのでdummyDataを用意しておく。
APIキーの取得手順
OpenWeatherMap（無料）
① https://openweathermap.org/api にアクセス
② 「Sign Up」で無料登録
③ ログイン後「API keys」タブからキーをコピー
④ 有効化まで最大数時間かかる場合がある
⑤ 使用するエンドポイント：https://api.openweathermap.org/data/2.5/weather

script.js（API通信関数）
// APIキー（実際には自分のキーを入れる） const API_KEY = "あなたのAPIキー"; const BASE_URL = "https://api.openweathermap.org/data/2.5/weather"; // 天気データを取得する関数 const fetchWeather = async (city) => { const url = `${{BASE_URL}}?q=${{encodeURIComponent(city)}}&units=metric&lang=ja&appid=${{API_KEY}}`; const res = await fetch(url); // 404 → 都市が見つからない if (res.status === 404) { throw new Error(`"${{city}}"は見つかりませんでした。都市名を確認してください。`); } // 401 → APIキーが不正 if (res.status === 401) { throw new Error("APIキーが無効です。確認してください。"); } if (!res.ok) { throw new Error(`通信エラー (${{res.status}})`); } return await res.json(); }; // レスポンスから必要なデータを整形する const formatWeather = (data) => ({ city: data.name, country: data.sys.country, temp: Math.round(data.main.temp), feelsLike:Math.round(data.main.feels_like), humidity: data.main.humidity, desc: data.weather[0].description, icon: `https://openweathermap.org/img/wn/${{data.weather[0].icon}}@2x.png`, wind: data.wind.speed, });
script.js（UI更新関数）
// DOM要素を取得 const el = { input: document.querySelector("#city-input"), btn: document.querySelector("#search-btn"), loading: document.querySelector("#loading"), error: document.querySelector("#error-msg"), result: document.querySelector("#weather-result"), cityName: document.querySelector("#city-name"), icon: document.querySelector("#weather-icon"), desc: document.querySelector("#weather-desc"), details: document.querySelector("#weather-details"), }; // ローディング状態を切り替える const setLoading = (isLoading) => { el.loading.hidden = !isLoading; el.btn.disabled = isLoading; if (isLoading) { el.error.hidden = true; el.result.hidden = true; } }; // エラーを表示する const showError = (message) => { el.error.textContent = message; el.error.hidden = false; }; // 天気データをUIに反映する const showWeather = (weather) => { el.cityName.textContent = `${{weather.city}}, ${{weather.country}}`; el.icon.src = weather.icon; el.desc.textContent = weather.desc; el.details.innerHTML = ` <div class="detail">🌡 気温: ${{weather.temp}}°C</div> <div class="detail">🤔 体感: ${{weather.feelsLike}}°C</div> <div class="detail">💧 湿度: ${{weather.humidity}}%</div> <div class="detail">💨 風速: ${{weather.wind}}m/s</div> `; el.result.hidden = false; };
