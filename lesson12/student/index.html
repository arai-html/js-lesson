async/awaitをマスターしよう
0〜25分
📖
async/awaitはPromiseをわかりやすく書ける構文です。現代のJS開発では必須の書き方です。
script.js（async/awaitの基本）
// 前回のPromiseをasync/awaitで書き直す // Promiseを返す関数（変更なし） const fetchData = () => { return new Promise((resolve) => { setTimeout(() => resolve({ name: "田中", age: 25 }), 1000); }); }; // .then()で書いた場合 fetchData().then(data => console.log(data)); // async/awaitで書いた場合（より読みやすい！） const getData = async () => { const data = await fetchData(); // Promiseが解決するまで待つ console.log(data); // {name:"田中", age:25} }; getData();
script.js（try/catchでエラーハンドリング）
const loadUser = async () => { try { // ローディング開始 document.querySelector("#loading").style.display = "block"; const data = await fetchData(); // 成功したらdataに入る console.log("取得成功：", data); } catch (error) { // エラーが起きたらここで受け取る console.log("エラー：", error.message); } finally { // 成功・失敗どちらでもローディングを解除 document.querySelector("#loading").style.display = "none"; } }; loadUser();
awaitはasyncの中でしか使えない
最重要ルール
await は async と宣言した関数の中でしか使えません。

✅ const fn = async () => {{ const data = await fetchData(); }}
❌ const data = await fetchData();（asyncなし → エラー）

「awaitを使いたい → async関数を作る」という流れで覚えましょう。

async関数の中だけ

2
fetchでAPIからデータを取得しよう
25〜45分
📖
fetchとasync/awaitを組み合わせて外部APIからデータを取得し、画面に表示します。
script.js（fetchの基本）
// fetch：URLにリクエストを送る const getUsers = async () => { try { // ① fetchでリクエスト（レスポンスオブジェクトが返る） const response = await fetch( "https://jsonplaceholder.typicode.com/users" ); // ② レスポンスが成功かチェック if (!response.ok) { throw new Error(`HTTPエラー: ${{response.status}}`); } // ③ JSONに変換（これも非同期なのでawaitが必要） const users = await response.json(); console.log(users); // ユーザーの配列 // ④ 画面に表示 const list = document.querySelector("#user-list"); list.innerHTML = users.map(user => ` <li> <strong>${{user.name}}</strong> <span>${{user.email}}</span> </li> `).join(""); } catch (error) { console.error("取得失敗：", error); document.querySelector("#user-list").innerHTML = "<li>データの取得に失敗しました</li>"; } }; getUsers();
💡
fetchの2段階のawaitを忘れずに。
① await fetch(url) → レスポンスオブジェクトを受け取る
② await response.json() → JSONを解析してJSオブジェクトに変換する

この2ステップは毎回必要です。セットで覚えましょう。
