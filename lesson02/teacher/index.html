変数を宣言しよう（const・let）
0〜25分
💬
講師がこう話す「変数はデータに名前をつけて保存する箱のようなものです。JavaScriptにはconst・let・varの3つの宣言方法がありますが、現代のJSでは基本的にconstを使い、後から値を変える必要があるときだけletを使います。varは古い書き方なので使いません。」
script.js
// const：値を変更できない（再代入不可） const name = "田中 太郎"; const age = 25; const isStudent = true; console.log(name); // "田中 太郎" console.log(age); // 25 console.log(isStudent); // true // let：値を変更できる（再代入可） let count = 0; console.log(count); // 0 count = count + 1; // 再代入OK console.log(count); // 1 // constに再代入しようとするとエラー // name = "山田 花子"; // ← TypeError！
constを基本にする理由
意図しない書き換えを防ぐ
constを使うと「この変数は変わらない」と宣言でき、コードの意図が明確になります。バグの原因になる「意図しない上書き」も防げます。

迷ったらまずconstで書いて、エラーが出たらletに変えるという進め方がおすすめです。
👆
講師メモvarを使わない理由（スコープの問題）は今は深入りしない。const/letの使い分けに集中させる。
script.js（命名規則）
// キャメルケース（小文字始まり・単語の区切りを大文字）が基本 const firstName = "太郎"; // ✅ キャメルケース const totalPrice = 1980; // ✅ const isLoggedIn = true; // ✅ 真偽値はis/hasで始めると分かりやすい // NG例 // const FirstName = "太郎"; // ← パスカルケース（クラス名に使う） // const first_name = "太郎"; // ← スネークケース（JS では使わない） // const a = "太郎"; // ← 意味不明な名前はNG

文字列・数値・テンプレートリテラル
25〜40分
💬
講師がこう話す「JavaScriptでは文字列の中に変数を埋め込む方法が3つあります。現代の書き方はテンプレートリテラル（バッククォート）を使う方法です。+で文字列をつなぐより読みやすく、改行も自然に書けます。」
script.js
const name = "田中 太郎"; const age = 25; const score = 88; // ① +で文字列を結合（古い書き方） console.log("名前：" + name + "、年齢：" + age); // ② テンプレートリテラル（現代の書き方）← バッククォートを使う console.log(`名前：${{name}}、年齢：${{age}}`); // 計算式も${{ }}の中に書ける console.log(`得点：${{score}}点 → 評価：${{score >= 80 ? "合格" : "不合格"}}`); // 数値と文字列の型変換 const numStr = "42"; // 文字列の"42" console.log(typeof numStr); // "string" const num = Number(numStr); // 数値に変換 console.log(typeof num); // "number" console.log(num + 10); // 52（数値として計算）
💡
バッククォート（`）はキーボードの Shift + @（Macの場合）または Shift + @（Windowsの場合）で入力できます。シングルクォート（'）と見た目が似ているので注意！
👆
講師メモバッククォートの入力場所がわからない生徒が多い。実際にキーボードを確認しながら一緒に入力させる。
script.js（よく使う文字列メソッド）
const text = " Hello, JavaScript! "; console.log(text.trim()); // 前後の空白を削除 console.log(text.toLowerCase()); // 小文字にする console.log(text.toUpperCase()); // 大文字にする console.log(text.length); // 文字数（スペース含む） console.log(text.includes("Java")); // "Java"が含まれるか → true console.log(text.replace("Hello", "Hi")); // 置換
