オブジェクトを作って操作しよう
0〜25分
📖
オブジェクトは関連するデータをキーと値のペアでまとめる仕組みです。実務で最もよく使うデータ構造です。
script.js（オブジェクトの基本）
// オブジェクトの作成（{}で囲み、キー: 値 のペアを書く） const user = { name: "田中 太郎", age: 25, isActive: true, email: "taro@example.com" }; // プロパティへのアクセス（ドット記法） console.log(user.name); // "田中 太郎" console.log(user.age); // 25 // プロパティへのアクセス（ブラケット記法）← 変数でキーを指定するとき const key = "name"; console.log(user[key]); // "田中 太郎" // プロパティの追加・変更・削除 user.phone = "090-0000-0000"; // 追加 user.age = 26; // 変更 delete user.isActive; // 削除 console.log(user);
script.js（オブジェクトの便利な操作）
const product = { name: "ノートPC", price: 98000, stock: 5 }; // キー・値・エントリーの一覧を取得 console.log(Object.keys(product)); // ["name","price","stock"] console.log(Object.values(product)); // ["ノートPC", 98000, 5] console.log(Object.entries(product)); // [["name","ノートPC"],...] // for...inでオブジェクトのキーをループ for (const key in product) { console.log(`${{key}}: ${{product[key]}}`); } // プロパティの存在チェック console.log("name" in product); // true console.log("color" in product); // false
オブジェクトの配列（最重要パターン）
実務で必ず出てくる
[{}, {}, {}] という「オブジェクトの配列」は実務で最もよく登場します。

例：[{{id:1, name:'りんご', price:200}}, {{id:2, name:'バナナ', price:100}}]

このパターンにmap/filterを組み合わせることで、商品リスト・ユーザー一覧・ToDoリストなどあらゆるUIを作れます。

分割代入・スプレッド構文・メソッド
25〜40分
📖
分割代入とスプレッド構文は現代JSの必須構文です。コードが短くスッキリ書けます。
script.js（分割代入）
const user = { name: "太郎", age: 25, city: "東京" }; // 従来の書き方 const name1 = user.name; const age1 = user.age; // 分割代入（オブジェクト）← 一気に取り出せる！ const { name, age, city } = user; console.log(name, age, city); // "太郎" 25 "東京" // 別名をつけて取り出す const { name: userName, age: userAge } = user; console.log(userName); // "太郎" // 配列の分割代入 const [first, second, third] = ["a", "b", "c"]; console.log(first); // "a" console.log(third); // "c"
script.js（スプレッド構文）
// スプレッド構文（...）：配列・オブジェクトを展開する // 配列のコピー・結合 const arr1 = [1, 2, 3]; const arr2 = [4, 5, 6]; const combined = [...arr1, ...arr2]; console.log(combined); // [1, 2, 3, 4, 5, 6] // オブジェクトのコピー・マージ const base = { name: "太郎", age: 25 }; const updated = { ...base, age: 26, city: "大阪" }; // ageは上書き、cityは追加される console.log(updated); // {name:"太郎", age:26, city:"大阪"} // 関数の引数として配列を展開 const nums = [3, 1, 4, 1, 5]; console.log(Math.max(...nums)); // 5
💡
スプレッド構文でオブジェクトをコピーするとき、const copy = original では参照コピー（同じオブジェクトを指す）になります。const copy = {{...original}} なら独立したコピーになります。これを知らないと意図しない書き換えが起きます。
